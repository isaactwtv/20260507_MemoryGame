# 記憶翻牌遊戲 (Memory Game)

一個基於 React + Vite + Tailwind CSS 開發的簡約記憶翻牌遊戲。

## 遊戲特色
- **限時挑戰**：在 45 秒內找出所有配對。
- **機會限制**：僅有 20 次翻牌機會。
- **獎勵機制**：每次成功配對可獲得額外 15 秒獎勵。
- **響應式設計**：支援行動裝置與桌面端。

## 本地開發

### 前提條件
- 已安裝 Node.js

### 安裝步驟

1. 安裝依賴：
   ```bash
   npm install
   ```

2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

3. 建立生產版本：
   ```bash
   npm run build
   ```

## 技術棧
- React 19
- Vite 6
- Tailwind CSS 4
- Lucide React (圖示)
- Framer Motion (動畫)
